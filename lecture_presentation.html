🎨 Frontend vs Backend vs Fullstack

🎨 Frontend (الواجهة الأمامية)

التعريف: كل شيء يتفاعل معه المستخدم

يشمل: الأزرار، القوائم، الألوان، التخطيط

التقنيات: HTML, CSS, JavaScript

⚙️ Backend (الخلفية)

التعريف: المنطق خلف الكواليس

يشمل: قواعد البيانات، المعالجة، الأمان

التقنيات: PHP, Laravel, MySQL

🔄 Fullstack (المطور الشامل)

التعريف: يقدر يشتغل Frontend و Backend

الميزة: فهم كامل للمشروع

💡 أمثلة تفاعلية

🔢 المسألة الأولى: الجمع

السؤال: لو عندك رقمين 5 و 3، إزاي تخلي الكمبيوتر يجمعهم؟

الحل: 5 + 3 = 8

المفهوم: المتغيرات والعمليات

🔒 المسألة الثانية: التحكم في الوصول

السؤال: إزاي تمنع شخص عمره 16 من دخول موقع +18؟

الحل: if (age < 18) { منع الدخول }

المفهوم: الشروط والتحكم المنطقي

🛒 المسألة الثالثة: المتجر الإلكتروني

السؤال: إزاي تخزن وتعرض المنتجات؟

الحل: قاعدة بيانات + صفحة عرض

المفهوم: قواعد البيانات وعرض المحتوى
