Building the Foundation of the Web

ü§î What is HTML?

HTML is the language we use to create web pages. Think of it as the bricks we use to build a website.

üìù Key Points:

  • HTML = HyperText Markup Language
  • It's NOT a programming language... it's a "markup" language that organizes page content
  • Every website you visit is built with HTML
  • HTML gives structure to web content

üè† Think of HTML like building a house:

HTML is like the frame and walls of a house - it gives structure. CSS is like the paint and decoration, and JavaScript is like the electrical system that makes things work!

üè∑Ô∏è What is a Tag?

<tagname>
Content
</tagname>

üéØ How to understand tags:

  • Opening tag: <p> - tells the browser "start a paragraph"
  • Content: "Any text" - what users see
  • Closing tag: </p> - tells the browser "end the paragraph"
<p>I am the first paragraph!</p>

Remember: Most tags need to be opened and closed, like opening and closing a door!

üìù 4 Essential Basic Tags

Tag Purpose Example
<h1> to <h6> Headings from big to small <h1>Big Title</h1>
<p> Paragraph of text <p>This is a paragraph</p>
<br> Line break (no closing tag) Line 1<br>Line 2
<hr> Horizontal line separator <hr>
<h1>Big Title</h1> <p>This is a paragraph</p> <hr> <p>This is a paragraph after the line</p>

üèóÔ∏è Complete HTML Page Structure

<!DOCTYPE html> <html> <head> <title>My First Page</title> </head> <body> <h1>Welcome!</h1> <p>I'm happy to be learning HTML</p> </body> </html>
Part Meaning
<!DOCTYPE html> Tells the browser this is an HTML page
<html> The big container that holds everything
<head> Information for the browser (not visible to users)
<title> Page name in the browser tab
<body> Everything the user sees

üìö Let's Review

‚ùì What is HTML?

A markup language for creating web pages

üè∑Ô∏è What is a tag?

Instructions wrapped in < > that tell the browser how to display content

‚úçÔ∏è How to write tags?

üìù Basic tags

üéØ Key Structure:

‚ùå Common Beginner Mistakes

üö´ Forgetting Closing Tags

<p>Hello World <h1>Title

‚úÖ Correct Way

<p>Hello World</p> <h1>Title</h1>

üí° Pro Tips:

  • Always close your tags (except self-closing ones like <br>)
  • Use proper indentation to make code readable
  • Test your HTML in different browsers
  • Don't forget the DOCTYPE declaration
